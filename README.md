# Mentor.AI# 🧠 Mentor AI

Your personal AI mentor built with Streamlit + Ollama.

Mentor AI analyzes:

* your daily schedule
* your goals
* your habits
* your check-ins

and gives:

* deep observations
* productive suggestions
* hidden opportunities
* warnings about bad habits
* long-term advice

---

# ✨ Features

* Smart mentor-style responses
* Point-wise advice
* Saves previous ideas
* Dark modern UI
* Runs locally
* Free and private
* Uses Ollama + Llama3

---

# 📦 Requirements

* Python 3
* Ollama
* Llama3 model

---

# 🚀 Installation

## 1. Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/mentor-ai.git
cd mentor-ai
```

---

## 2. Create Virtual Environment

```bash
python3 -m venv venv
```

Activate:

### Linux / Mac

```bash
source venv/bin/activate
```

### Windows

```bash
venv\Scripts\activate
```

---

## 3. Install Dependencies

```bash
pip install streamlit ollama
```

---

## 4. Install Ollama

Download:

https://ollama.com

---

## 5. Download Llama3

```bash
ollama run llama3
```

---

# ▶️ Run Mentor AI

Start Ollama first:

```bash
ollama run llama3
```

Open another terminal:

```bash
source venv/bin/activate
python3 -m streamlit run app.py
```

---

# 🌐 Open In Browser

```txt
http://localhost:8501
```

---

# 📱 Use On Phone

If phone and laptop are on same WiFi:

Open the Network URL shown in terminal:

```txt
http://192.168.x.x:8501
```
